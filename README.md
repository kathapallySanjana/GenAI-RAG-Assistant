# GenAI RAG Assistant

This project is a **Retrieval-Augmented Generation (RAG) chatbot** built using **Python, Flask, and OpenAI embeddings**.
It retrieves relevant information from documents and provides answers through a simple chat interface.

## Features

* Document embedding using OpenAI embeddings
* Retrieval of relevant document chunks
* Flask backend API
* Simple chat interface
* Context-based answers using RAG

## Project Structure

GenAI-RAG-Assistant
│
├── app.py
├── docs.json
├── embeddings.py
├── rag.py
├── requirements.txt
└── index.html

## Installation

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open the application in your browser:

http://127.0.0.1:5000/
